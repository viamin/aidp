# Port Configuration for AIDP Development

This document lists all ports used by this project's development environment.

## Forwarded Ports

The following ports are automatically forwarded when running in a devcontainer:

| Port | Service | Description |
| ------ | --------- | ------------- |
| 3000 | Web Application | Main application server |
| 5432 | PostgreSQL | Database server |

## Port Details

### Port 3000 - Web Application

- **Protocol**: HTTP
- **Purpose**: Main Rails/Sinatra application server
- **Access**: <http://localhost:3000>

### Port 5432 - PostgreSQL

- **Protocol**: TCP
- **Purpose**: PostgreSQL database server
- **Connection**: `postgresql://localhost:5432/database_name`

## Custom Ports

To add custom ports to your development environment:

1. Edit `.aidp/aidp.yml`:

   ```yaml
   devcontainer:
     manage: true
     custom_ports:
       - number: 8080
         label: "Custom Service"
   ```

2. Apply the changes:

   ```bash
   aidp devcontainer apply
   ```

## Port Conflicts

If you encounter port conflicts:

1. Check which process is using the port:

   ```bash
   lsof -i :3000
   ```

2. Stop the conflicting process or change your application's port configuration

3. Update your devcontainer configuration if needed
